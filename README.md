🎯 Interview Yodha

An AI-powered full-stack mock interview platform designed to help students and job seekers prepare for technical, HR, and behavioral interviews with real-time personalized feedback.

🌐 Live Demo: https://inter-view-yodha.vercel.app

📌 About The Project

Interview Yodha is a modern AI-driven interview preparation platform built as our B.Tech Final Year Project.

The platform bridges the gap between theoretical knowledge and real-world interview readiness by simulating realistic interview environments using AI.

Unlike traditional platforms that only provide coding questions or static theory, Interview Yodha offers:

🤖 AI mock interviews
📄 Resume assistance
📊 Performance analytics
📈 Progress tracking
🔐 Secure authentication
📱 Fully responsive UI

The goal is simple: Help students become confident, interview-ready, and industry prepared.

✨ Features
🎤 AI Mock Interviews
📊 Performance Dashboard
📄 Resume Support
🔐 Authentication with Clerk
📱 Responsive Design
🛠️ Tech Stack
Frontend: Next.js, Tailwind CSS, Shadcn UI
Backend: Next.js API Routes
Database: PostgreSQL (Neon DB)
ORM: Prisma
Authentication: Clerk
AI: Gemini API
Background Jobs: Inngest
Deployment: Vercel
